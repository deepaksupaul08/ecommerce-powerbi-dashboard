# 📊 Olist E-Commerce Analytics Dashboard

## Overview

This project presents an interactive Power BI dashboard built using the Olist Brazilian E-Commerce dataset. The dashboard provides insights into sales performance, customer behavior, product performance, and delivery operations through a set of business-focused visualizations and KPIs.

The objective of this project is to transform raw e-commerce data into actionable business insights that support data-driven decision-making.

---

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Power Query
* GitHub

---

## Dataset

The project uses the Olist Brazilian E-Commerce Dataset, which contains information related to:

* Orders
* Customers
* Products
* Payments
* Sellers
* Reviews
* Delivery Information

## Key Business Insights

* Revenue is concentrated in a limited number of high-performing states.
* Customer activity varies significantly across regions.
* Certain product categories generate a disproportionate share of total revenue.
* Delivery performance differs by geographic location.
* Freight costs have a measurable impact on operational efficiency.
